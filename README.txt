Alive5 - SMS Made Simple.

== Description ==

Alive5 is the easy-to-use SMS Chat solution for your WordPress website. Visitors can immediately sms chat session with someone at your company who can answer their questions, in real-time.
This WordPress plug-in instantly updates your WordPress site with Alive5 Tracking Code which tracks visitors in real-time and also displays an sms chat icon.

== Installation ==

Installation Instructions:
1. From your Wordpress Admin Dashboard, go to ‘Plugins’ and “Add New.”
2. In the search bar, type in “Alive5” and then click the Search Plugins button.
3. You will see the “Alive5” plugin displayed. Click the “Install Now” link.
4. Wordpress will ask you if you are sure that you want to download the plugin. Click Yes.
5. Wordpress will install the plugin, then you will need to click the “Activate Now” link.

Configuration Settings:
Now that you have installed the Alive5 plugin, you are just minutes away from chatting
with your site visitors via sms! To configure your Alive5 plugin, click on the Alive5 in Admin dashboard and then click on the Settings tab. Provide the alive5 credentials and click next.

New Account Setup:
1. If you are new to Alive5, visit http://alive5.com.
2. Create a organization, and input your email address. Then click the “Register” button
3. You will receive an email, verify the information for account creation and then please go to http://app.alive5.com and login as Admin.
